# Infrastructure & Deployment (`deployment/`)

## Purpose
Houses Docker container definitions, Terraform IaC manifests, CloudWatch/Catalyst monitoring dashboards, and central logging configurations for production environments.

## Owner
- **Primary Owner**: Member 4 (M4 - Platform & Real-Time)

## Structure
- `docker/`: Base container images and dev containers.
- `terraform/`: Infrastructure provisioning scripts.
- `monitoring/`: Alerting and health check rules.
- `logging/`: Log formatters and ingestion agents.
