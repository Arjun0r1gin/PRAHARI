# Changelog

All notable architectural updates and feature additions to the **PRAHARI** project will be documented in this file.

## [v0.4] - 2026-07-22
### Added
- Real-time simulation stream engine (`simulation/`) and event feed adapter.
- Playback controls and live feed screens in M3 frontend and M4 platform backend.

## [v0.3] - 2026-07-22
### Added
- Hotspot Detection Engine (`hotspot-engine/`) backing spatial analytics.
- Network Graph Analysis and centrality model artifacts.

## [v0.2] - 2026-07-22
### Added
- Multi-station Data Fusion Engine (`data-fusion/`) with support for 4 heterogeneous police station feeds.
- Unified Record Schema definition (`unified-record-schema.js`).

## [v0.1] - 2026-07-22
### Added
- Initial PRAHARI platform repository architecture, Catalyst function structure, and statewide dataset layout.
