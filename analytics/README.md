# Analytics & Algorithm Research (`analytics/`)

## Purpose
Houses experimental statistical modeling, network analysis algorithms, cross-district pattern correlation, time-series forecasting, and spatial clustering code prior to production deployment.

- Normalized historical crime records from `data/districts/`.

## Outputs
- Trained model weights and validated algorithmic strategies exported to `models/`.

## Dependencies

- Python scientific stack (Scikit-Learn, NetworkX, GeoPandas)
